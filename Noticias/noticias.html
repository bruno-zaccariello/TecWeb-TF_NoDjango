Alunos do curso de sistema da informação, desenvolvem um sistema para escolas

Alunos do quarto semestre do curso de sistema da informação, desenvolvem um sitema para escolas
públicas, os mesmos disseram que pode ser um novo método de ampliar a organização das nossas escolas, com um sistema mais fluído e dinâmico para melhor adaptação a essa nova era tecnológica que estamos adentrando atualmente. O sistema possuí diversas funcionalidades desde um login altamente seguro assim como uma área para perguntas, para com o pessoal da secretária da unidade a qual o usuário escolher acessar.
